# PolyNews SEO & Keyword Strategy

## Keyword Research Summary

### Primary Target Keywords (High Volume, Moderate Competition)
These are the keywords PolyNews should rank for to capture mainstream audience:

```
1. "prediction markets" - 880 searches/month
   - Intent: Informational (how do they work)
   - Difficulty: Medium
   - Search type: Desktop + Mobile
   - CTR: 5-7%
   - Priority: CRITICAL

2. "Bitcoin price prediction" - 3,600 searches/month
   - Intent: Commercial/Informational
   - Difficulty: High (but worth it)
   - Search type: Mostly mobile
   - CTR: 3-5%
   - Priority: HIGH

3. "election prediction odds" - 1,200 searches/month
   - Intent: Informational
   - Difficulty: Medium-High
   - Search type: Desktop
   - CTR: 6-8%
   - Priority: HIGH

4. "crypto market forecast" - 2,100 searches/month
   - Intent: Informational/Commercial
   - Difficulty: High
   - Search type: Mobile
   - CTR: 4-6%
   - Priority: HIGH

5. "what will happen next week" (+ topic variants) - 5,400/month
   - Intent: Informational
   - Difficulty: Low
```

## Content Optimization Templates

### Blog Post Meta Tag Template
```
Title: [Keyword] - [Value Prop] | PolyNews
(Keep under 60 characters; use primary keyword at start)

Examples:
- "Bitcoin Price Prediction 2025 | Polymarket Consensus"
- "Election Odds: Real-Time Prediction Markets | PolyNews"
- "Recession Prediction: What Markets Say | PolyNews"

Meta Description: [Main point] + [CTR hook] + [60-character max]

Examples:
- "See what prediction markets say about [keyword]. Updated live with
  real trading data and crowd wisdom."
- "Polymarket traders betting [X]% on [event]. Real-time odds with
  [benefit] to traders."
```

### H1 Optimization
```
Pattern: "[Keyword] - [Angle]"

Examples:
- "Bitcoin Price Prediction: What Markets Currently Show"
- "Election Prediction Markets: Why They're Accurate"
- "Recession Prediction: Market Consensus Explained"

Rule: Always include target keyword in H1
Rule: Always include benefit or angle
Rule: Keep under 70 characters when possible
```

### Introduction Paragraph Template
```
"[Context about topic]. According to Polymarket prediction markets,
[finding/stat]. This guide explores [what readers will learn],
breaking down [key concept] and showing how [benefit] for [audience]."

Word count: 80-120 words
Must include: keyword (1-2x), value prop, audience clarification
```

### Section Heading Pattern
```
H2: "[Keyword Component] - [Benefit/Angle]"

Examples for "Bitcoin prediction" post:
- "What Prediction Markets Say About Bitcoin (vs. Traditional Analysis)"
- "Bitcoin Price Targets - How Markets Determine Odds"
- "Trading Bitcoin Predictions - Market Mechanics Explained"

Rule: Each H2 should target related long-tail keyword
Rule: Use 4-6 H2s per 1,500-word post
```

## Technical SEO Checklist

### Per-Blog Post
- [ ] Title tag (50-60 chars, keyword included)
- [ ] Meta description (150-160 chars, CTA included)
- [ ] H1 (unique per page, includes keyword)
- [ ] H2-H4 hierarchy (logical structure)
- [ ] Image alt text (descriptive, keyword where relevant)
- [ ] Internal links (3-5 per post, descriptive anchors)
- [ ] Readability (short paragraphs, active voice)
- [ ] Word count (1,000-2,000 for competitive keywords)
- [ ] Loading speed (<3 sec on mobile)
- [ ] Mobile responsiveness (test on devices)
- [ ] Outbound links (2-3 to authoritative sites)
- [ ] Engagement elements (images, charts, videos)

### Per-Market Page
- [ ] Title includes probability + market statement
- [ ] Meta description includes key metrics
- [ ] H1 = market statement
- [ ] Related markets internal links
- [ ] Breadcrumb navigation

- [ ] Social sharing buttons

## Competitor Analysis

### Key Competitors to Monitor
1. **Metaculus.com** - Prediction platform, strong SEO for "prediction market"
2. **PredictIt.org** - Political prediction market, ranks for election keywords
3. **Hypermind** - General prediction platform
4. **CoinGecko/CoinMarketCap** - Crypto price prediction sections
5. **TradingView** - Trading and forecasting platform
6. **FiveThirtyEight** - Election forecasting authority (traditional but ranks well)

### Competitive Advantage Angles
1. **Real-time odds** - Update faster than competitors
2. **Broad category coverage** - More categories than most
3. **Accuracy tracking** - Historical data others lack
4. **Explanation quality** - Make complex concepts understandable
5. **API access** - Data service others don't offer
6. **User engagement** - Tracking, voting, saving features
